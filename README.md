# File Archiving System

A secure web-based application designed to efficiently store, organize, compress, and retrieve digital documents and files. This system helps individuals or organizations manage data lifecycle, free up active storage space, and maintain long-term digital records systematically.

## 🚀 Features
* **File Upload & Categorization:** Easily upload multi-format files and organize them into specific categories or folders.
* **Compression & Archiving:** Automated or manual bundling of old or large files into compressed formats to save storage space.
* **Advanced Search & Retrieval:** Quick filter and search functionality based on file names, categories, tags, or upload dates.
* **Access Control:** User authentication to ensure that only authorized personnel can view, download, or delete archived files.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL (via XAMPP phpMyAdmin)

## 💻 How to Run the Project
1. Download or clone this repository as a `.zip` file.
2. Extract the folder and move it into your XAMPP installation directory: `C:\xampp\htdocs\`
3. Open the **XAMPP Control Panel** and start both **Apache** and **MySQL**.
4. Open your browser and go to `http://localhost/phpmyadmin/` to create a new database.
5. Import your project's `.sql` database file (if included) into phpMyAdmin.
6. Open a new tab in your browser and visit: `http://localhost/file_archiving_system/`
